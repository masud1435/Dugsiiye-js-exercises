if Statement

Definition: The if statement is used to execute a block of code if a specified condition is true.

Syntax:

if (condition) {
// code to be executed if the
condition is true
}

if-else Statement

Definition: The if-else statement is used to execute one block of code if a condition is true, and another block of code if the condition is false.

Syntax:

if (condition) {
// code to be executed if the
condition is true
} else {
// code to be executed if the
condition is false
}
